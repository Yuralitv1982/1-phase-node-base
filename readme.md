# Phase 1: Architectural Foundation (Node.js)

Добро пожаловать в репозиторий Фазы 1. Здесь закладывается фундамент для архитектора Node.js. Мы не просто учим синтаксис — мы изучаем внутреннее устройство систем.

## 🎯 Цель Фазы
Дать фундаментальные знания, необходимые для глубокого понимания Node.js и смежных дисциплин. Мы идем от «железа» и архитектуры компьютера к высокоуровневой логике приложений.

## 📜 Принципы обучения
- **Single-threaded**: Одна задача в фокусе.
- **Инженерный подход**: Никакой магии, только логика и понимание работы под капотом.
- **Моделирование**: Мы не зазубриваем, мы моделируем ситуации (сбои, нагрузки, потоки).
- **Daily Commit**: Нет коммита — день прошел зря. Это доказательство твоей работы.
